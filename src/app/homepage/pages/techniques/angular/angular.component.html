Angular

Serve Angular application with Nest.

Basic example

Serving Angular is possible via catching NotFoundException and serve Angular dist/index.html on non existed routes: 

{{ notFoundExceptionFilter }}

Global filter

Then, we need to register global filter in main.ts: 

{{ mainTs }}

If a request to the server hits a non existed route, the NotFoundException fired, then NotFoundExceptionFilter catch the NotFoundException and serve dist/index.html
